Выбор режима расчёта

Выберите формат работы: быстрый ориентир, основной A4, ведомость сделок или расширенный расчёт периода.

На старт A4 Ведомость Простой Расширенный

Простой расчёт

Быстрая оценка экономики офиса без детального ввода каждого агента и каждой сделки.

  • общий оборот;
  • количество агентов;
  • расходы;
  • быстрый ориентир.
Открыть простой режим

Основной A4-калькулятор

Основной управленческий расчёт текущего месяца по агентам, сделкам, расходам, роялти и мотивациям.

  • агенты и сделки;
  • выплаты и рефералы;
  • роялти;
  • итог собственнику.
Открыть A4

Ведомость сделок

Табличная ведомость для Генерального: сделки строками, итоги агента и общий итог офиса.

  • реестр сделок;
  • итоги агента;
  • контроль выплат;
  • алиас через table.html.
Открыть ведомость Прямой вход

Расширенный расчёт

Расчёт периода для собственника: месяц, квартал, полугодие или год с агентами, сделками и черновиком.

  • период 1 / 3 / 6 / 12 месяцев;
  • быстрый или подробный ввод;
  • мотивации и обязательства;
  • сохранение черновика.
Открыть расширенный режим

Как выбрать режим

Для текущего месяца используйте A4. Для сделки-по-строке откройте ведомость. Для периода больше месяца используйте расширенный режим.

Расчёт является управленческой моделью и помогает оценить экономику офиса.
