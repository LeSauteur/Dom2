Навигатор режимов

Домиан — выбор режима расчёта

Выберите формат работы под задачу: быстрый ориентир, точный расчёт текущего месяца или будущий расчёт периода.

Быстрая оценка

Простой режим для грубой оценки экономики офиса без подробного ввода каждого агента и каждой сделки.

  • общий оборот;
  • количество агентов;
  • средняя схема выплат;
  • расходы;
  • быстрый диагноз.
Открыть simple.html

Точный расчёт месяца

Основной A4-калькулятор для расчёта текущего месяца по агентам, сделкам, расходам, роялти и мотивациям.

  • агенты;
  • сделки;
  • выплаты;
  • роялти;
  • мотивации;
  • итог собственнику.
Открыть index.html

Расчёт периода

Будущий расширенный режим для месяца, квартала, полугодия или года. Сохранение черновика и календарная логика будут добавлены отдельными этапами.

  • месяц / квартал / полугодие / год;
  • общий или подробный ввод;
  • месячные сводки;
  • будущие обязательства;
  • черновики позже.
Открыть extended.html

Как выбрать режим

Если нужно быстро понять порядок цифр — используйте быструю оценку.

Если нужно точно посчитать текущий месяц — используйте A4.

Если нужен период, календарь мотиваций или черновики — используйте расчёт периода.
